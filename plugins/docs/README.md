# Docs

Skills, agents, and commands for documentation and document generation.

## Skills

| Skill | Description |
|-------|-------------|
| `api-documentation-generator` | Generate API docs from code, OpenAPI specs, or endpoints |
| `docx` | Generate and manipulate Word documents |
| `pdf` | Generate and manipulate PDF documents |
| `pptx` | Generate and manipulate PowerPoint presentations |
| `xlsx` | Generate and manipulate Excel spreadsheets |

## Agents

| Agent | Description |
|-------|-------------|
| `technical-writer` | Documentation specialist for READMEs, API docs, and guides |

## Commands

| Command | Description |
|---------|-------------|
| `setup-docs` | Initialize documentation structure for a project |

## Usage

Use the technical-writer agent for documentation tasks. The document format skills (docx, pdf, pptx, xlsx) handle file generation.

```
Load skill api-documentation-generator
Load skill pdf
```
