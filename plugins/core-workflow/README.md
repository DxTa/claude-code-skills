# Core Workflow

Essential workflow skills, agents, and commands for structured development processes.

## Skills

| Skill | Description |
|-------|-------------|
| `code-review` | Two-stage review process: spec compliance first, then code quality |
| `planning-with-files` | Task plans and notes file management with session-based naming |
| `push-all` | Quality gates and checks before pushing to remote |
| `skill-creator` | Create new skills following the standard skill template |
| `spec-analyzer` | Socratic spec enhancement through targeted questioning |

## Agents

| Agent | Description |
|-------|-------------|
| `self-reflect` | Plan validation and edge case identification for T3+ tasks |
| `software-architect` | Architecture guidance and system design review |

## Commands

| Command | Description |
|---------|-------------|
| `setup-repo` | Initialize repository structure and configuration |
| `spec-analyze` | Run the spec analyzer against requirements |
| `push-all` | Push changes with pre-push quality gates |

## Usage

Core workflow is the foundation for all development tasks. The planning and review skills are mandatory for T2+ tiers.

```
Load skill planning-with-files
Load skill code-review
```
