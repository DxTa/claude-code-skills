# Backend

Skills and agents for backend development, API design, and database management.

## Skills

| Skill | Description |
|-------|-------------|
| `backend-development` | Backend architecture, API design, and server-side patterns |
| `databases` | Database design, query optimization, and data modeling |

## Agents

| Agent | Description |
|-------|-------------|
| `backend-specialist` | Specialist agent for API, database, and auth implementation |

## Usage

Use the backend-specialist agent for implementation tasks involving APIs, databases, or authentication flows.

```
Load skill backend-development
Load skill databases
```
