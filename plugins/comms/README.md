# Comms

Skills for communication platforms, calendar, email, and repository operations.

## Skills

| Skill | Description |
|-------|-------------|
| `gcal` | Google Calendar integration for event management |
| `gdrive` | Google Drive file operations and document access |
| `github-operations` | GitHub issue, PR, and repository management workflows |
| `glab-operations` | GitLab issue, MR, and repository management workflows |
| `gmail` | Gmail integration for email composition and search |
| `internal-comms` | Internal communication workflows and templates |

## Usage

Communication skills connect to external services. Ensure required API credentials or MCP servers are configured before use.

```
Load skill github-operations
Load skill gmail
```
