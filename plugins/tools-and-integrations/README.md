# Tools and Integrations

Skills for browser tooling and MCP server development.

## Skills

| Skill | Description |
|-------|-------------|
| `chrome-devtools` | Chrome DevTools automation, debugging, and profiling |
| `mcp-builder` | Build and configure Model Context Protocol (MCP) servers |

## Usage

Use chrome-devtools for browser-based debugging and performance analysis. Use mcp-builder to create new MCP server integrations.

```
Load skill chrome-devtools
Load skill mcp-builder
```
