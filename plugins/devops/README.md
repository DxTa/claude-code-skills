# DevOps

Skills, agents, and commands for CI/CD, infrastructure, and cloud operations.

## Skills

| Skill | Description |
|-------|-------------|
| `ansible-playbook-creator` | Generate and manage Ansible playbooks for automation |
| `ci-cd-pipeline-builder` | Design and configure CI/CD pipelines |
| `cloud-cost-optimizer` | Analyze and reduce cloud infrastructure costs |
| `infrastructure-as-code-generator` | Generate IaC templates (Terraform, CloudFormation, CDK) |

## Agents

| Agent | Description |
|-------|-------------|
| `devops-engineer` | CI/CD, Docker, Kubernetes, and cloud infrastructure specialist |

## Commands

| Command | Description |
|---------|-------------|
| `setup-ci-cd` | Initialize CI/CD pipeline configuration for a project |

## Usage

Use the devops-engineer agent for infrastructure tasks. Combine with the `aws-cdk-development` skill for CDK-specific work.

```
Load skill ci-cd-pipeline-builder
Load skill infrastructure-as-code-generator
```
