# Design

Skills for visual design, creative coding, and brand-consistent asset creation.

## Skills

| Skill | Description |
|-------|-------------|
| `algorithmic-art` | Generate algorithmic and generative art with code |
| `artifacts-builder` | Build visual artifacts, diagrams, and interactive elements |
| `brand-guidelines` | Apply and enforce brand identity across assets |
| `canvas-design` | Canvas-based design and layout composition |
| `theme-factory` | Generate consistent color themes and design tokens |

## Usage

Design skills are standalone and do not require specific agents. They can be combined for comprehensive design workflows.

```
Load skill theme-factory
Load skill brand-guidelines
```
