# Frontend

Skills and agents for frontend development, UI design, and web framework usage.

## Skills

| Skill | Description |
|-------|-------------|
| `frontend-design` | UI/UX design principles and component architecture |
| `frontend-development` | Frontend development guidelines and best practices |
| `ui-styling` | CSS, styling systems, and visual consistency |
| `web-artifacts-builder` | Build interactive web artifacts and prototypes |
| `web-frameworks` | Framework-specific patterns for React, Vue, and others |

## Agents

| Agent | Description |
|-------|-------------|
| `frontend-specialist` | Specialist agent for React, Vue, CSS, and UI implementation |

## Usage

Use the frontend-specialist agent for implementation tasks. Load individual skills for specific guidance.

```
Load skill frontend-development
Load skill ui-styling
```
