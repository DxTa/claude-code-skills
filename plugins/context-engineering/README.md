# Context Engineering

Claude Code-adapted skills for task orchestration, context management, and workflow governance.

> **Note:** These are CC-adapted versions of the context engineering skills, tailored for Claude Code's tool and agent model.

## Skills

| Skill | Description |
|-------|-------------|
| `master-checklist` | Comprehensive task workflow with tier-based gates and checkpoints |
| `anti-patterns` | Common mistakes and pitfalls to avoid during development |
| `agent-selection` | Decision tree for choosing the right agent or tool for a task |
| `token-management` | Context window optimization and token budget strategies |

## Usage

These skills govern how tasks are planned, executed, and completed. The master checklist is the central reference for all tier-based workflows.

```
Load skill master-checklist
Load skill agent-selection
```
