# Media and Data

Skills for media processing, data management, and knowledge organization.

## Skills

| Skill | Description |
|-------|-------------|
| `media-processing` | Audio, video, and image processing pipelines |
| `slack-gif-creator` | Create and post GIF animations to Slack |
| `memvid` | Video-based memory and knowledge capture |
| `pkm` | Personal knowledge management and note organization |

## Usage

These skills handle media transformation and data organization tasks. Some may require external tools (ffmpeg, imagemagick) to be installed.

```
Load skill media-processing
Load skill pkm
```
