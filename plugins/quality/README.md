# Quality

Skills, agents, and commands for testing, code review, and quality assurance.

## Skills

| Skill | Description |
|-------|-------------|
| `chaos-engineering-toolkit` | Fault injection and resilience testing strategies |
| `code-review-plugin` | Automated code review checks and quality metrics |
| `performance-test-suite` | Load testing, benchmarking, and performance profiling |
| `pr-review-toolkit` | Pull request review checklists and feedback templates |
| `test-coverage-analyzer` | Coverage analysis and gap identification |
| `webapp-testing` | End-to-end, integration, and unit testing for web apps |

## Agents

| Agent | Description |
|-------|-------------|
| `qa-engineer` | Test strategy, coverage analysis, and quality validation |
| `code-simplifier` | Simplify code structure while preserving functionality |

## Commands

| Command | Description |
|---------|-------------|
| `review` | Run a two-stage code review on changes |
| `setup-tests` | Initialize test infrastructure for a project |

## Usage

Quality skills are mandatory for T2+ workflows. The code-simplifier agent runs before review and handoff.

```
Load skill webapp-testing
Load skill test-coverage-analyzer
```
