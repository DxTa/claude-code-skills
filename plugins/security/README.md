# Security

Skills and agents for security auditing, vulnerability scanning, and secure development.

## Skills

| Skill | Description |
|-------|-------------|
| `security-pro-pack` | OWASP-based security review, threat modeling, and hardening |
| `security-test-scanner` | Automated security scanning and vulnerability detection |

## Agents

| Agent | Description |
|-------|-------------|
| `security-engineer` | Security specialist for audits, OWASP compliance, and crypto review |

## Usage

Use the security-engineer agent for security audits and threat assessments. The skills provide structured checklists and scanning procedures.

```
Load skill security-pro-pack
Load skill security-test-scanner
```
